블로그 메인 화면에 있는 이미지를 불러올 때 깜빡거리는 이슈가 발생했다.
이를 해결하기 위해 next/image 에서 제공하는 property인 placeholder=’blur’와 blurDataURL를 사용하였다. 그 결과 이미지 로딩 시 빈 이미지 공간이 블러처리 되어 기존보다 보기 편해졌다.

```bash
<Image
  src="/jump.gif"
  width={850}
  height={300}
  alt="Picture of the author"
  placeholder="blur" //추가
  blurDataURL="data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAAAAoAAAAKCAYAAACNMs+9AAAAFklEQVR42mN8//HLfwYiAOOoQvoqBABbWyZJf74GZgAAAABJRU5ErkJggg==" // 추가
/>
```

여기서 궁금한 점이 생겼다. 왜 blurDataURL에서 base64 인코딩을 사용했는지, base64 인코딩 과정에 대해서는 추가로 공부해봐야겠다. 🤔
